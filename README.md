# Wireshark-
The initial data was obtained using the Wireshark sniffer programme on the city's backbone network. Over a period of five hours, 278,557 packets were tracked, including: 158 TCP (Transmission Control Protocol); 493 Address Resolution Protocol; 25,733 MPEG; 250,242 User Datagram Protocol and others. The share of UDP protocol packets was 89.83%.
